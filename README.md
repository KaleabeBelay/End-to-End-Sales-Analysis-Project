# 🛒 End-to-End Sales Analysis Project

## 📌 Overview
Analyzed 3,900 customer transactions to uncover insights into spending behavior, customer segments, and product performance to support data-driven decisions.

## 📊 Key Steps
- Data cleaning & preparation using Python (pandas)  
- Data analysis using SQL (SQL Server)  
- Dashboard development in Power BI  

## 🔍 Key Insights
- Identified key customer segments (New, Returning, Loyal)  
- Analyzed revenue by demographics and subscription status  
- Determined top-performing and discount-driven products  
- Evaluated impact of shipping and promotions on spending  

## 💡 Recommendations
- Strengthen subscription and loyalty programs  
- Optimize discount strategies  
- Prioritize high-performing products in marketing  

## 🛠️ Tech Stack
Python | SQL Server | Power BI
